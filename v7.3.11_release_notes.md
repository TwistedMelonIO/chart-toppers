# v7.3.11 Release Notes — R3 mashup list capped at 56px

**Released: May 2, 2026**

## What's Changed
- **R3 mashup list font on the stage host iPad now caps at 56px** (was 112px). Operator wanted 80s/2000s mashups rendered smaller to match how 1990s already auto-shrank to fit. Lists that still don't fit at 56px will continue to auto-shrink down to 22px.

## Technical Changes
- `public/stagehost.js`: `showR3List()` now calls `fitAnswerTrack(56)` instead of the default `fitAnswerTrack()` (112).
