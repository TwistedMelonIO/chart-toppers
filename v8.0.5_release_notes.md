# Chart Toppers v8.0.5

## Animations disabled on the scoring dashboard and settings page

All `animation:` declarations in `public/styles.css` and the inline styles of `public/index.html`, `public/settings.html` are commented out — the dashboard and settings pages no longer shimmer, pulse, fade, or flash. Plain static UI.

The animations had been triggering an unexpected flashing effect on at least one show machine and the team needed it gone immediately. The `@keyframes` definitions are left intact (just unused) so they can be re-enabled later by uncommenting the `animation:` lines.

`public/stagehost.css` is untouched — the stage-host iPad is unaffected.

## Files changed
- `public/styles.css` — 25 `animation:` declarations commented out
- `public/index.html`, `public/settings.html` — inline `animation:` declarations commented out
- `package.json`, `public/index.html`, `public/settings.html` — version bump to 8.0.5
