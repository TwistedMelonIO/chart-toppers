# v5.2.0 Release Notes — Always-Export License Certificate

**Released: April 25, 2026**

## What's New

### Always-export `.docforge` certificate file
- **Summary**: The license installer now always writes a `.docforge` certificate file to the Desktop after a successful activation.
- **Details**: Removed the `Generate .docforge file? (y/N)` confirmation gate from `install_license.sh`. After a successful activation, the script prompts for the licensee name and expiry days (both optional — Enter to skip), then unconditionally writes `~/Desktop/chart-toppers-license.docforge` containing the machine ID, license key, licensee, and expiry metadata.
- **Impact**: Operators always end up with a portable certificate file they can hand off to Docforge for cert generation, with no chance of accidentally skipping the prompt.

## Technical Changes
- `install_license.sh`: dropped the y/N gate around the docforge export; the file is now written on every successful activation.
