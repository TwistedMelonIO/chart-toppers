# v4.0.0 Release Notes — Chart Toppers

**Released: April 3, 2026**

## What's New in v4

### Golden Record Power-Up
Each team gets one Golden Record per game. Activated via Companion, the next correct answer in Rounds 1-3 earns double seconds. Dashboard shows armed/used state with animated gold indicator. QLab cues `GRA`/`GRI` fire on activation.

### Bundled QLab Buzzer
USB buzzer support ships with Chart Toppers. Runs as a macOS background service — starts on login, restarts on crash, no Terminal window needed. Dashboard footer shows real-time buzzer connection status. Buzzer triggers logged to activity log.

### One-Command Installer
Single install script handles everything: Homebrew, Python 3.12, Docker containers, buzzer virtual environment, launchd service, Accessibility permissions, and license activation. Techs run one command on a fresh Mac.

### Round 4 Track Control
Companion triggers Round 4 tracks via OSC (`/chart-toppers/r4/1` to `/r4/4`). Each track can only be played once — replays rejected by server. State resets on round change or game reset.

### QLab Enhancements
- Round 1 cues labelled `[SHORT]` and `[LONG]` for easy re-triggering
- Round 1 short clips include `Correct Answer: Artist - Track` in QLab notes
- Reset All now arms all cues in the QLab workspace

### Start/Stop Scripts
- `./start.sh` — launches Docker + buzzer together
- `./stop.sh` — cleanly shuts everything down

## OSC Command Reference

| Command | Action |
|---|---|
| `/chart-toppers/correct/anthems` | Correct answer for Anthems |
| `/chart-toppers/correct/icons` | Correct answer for Icons |
| `/chart-toppers/golden-record/anthems` | Activate Golden Record |
| `/chart-toppers/golden-record/icons` | Activate Golden Record |
| `/chart-toppers/r4/1` to `/r4/4` | Play Round 4 track (one-shot) |
| `/chart-toppers/playing/anthems` | Team playing |
| `/chart-toppers/playing/icons` | Team playing |
| `/chart-toppers/reset` | Reset all teams + arm QLab cues |
| `/chart-toppers/round/1` to `/round/4` | Set active round |
