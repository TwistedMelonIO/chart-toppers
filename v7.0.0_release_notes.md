# v7.0.0 Release Notes — Round 4 Pity Gift

**Released: May 2, 2026**

## What's New

### Round 4 Zero-Score Pity Gift
- **Summary**: Teams entering Round 4 with 0 earned seconds now receive a 10-second pity gift.
- **Details**: On R4 entry, any team with `earnedTime === 0` is bumped to 10s. The QLab countdown video, dashboard timer, and scoring all see consistent state.
- **Impact**: Zero-score teams still get a real R4 moment instead of stalling, and the dashboard countdown no longer drifts to ~90s while QLab plays something else.

## Bug Fixes
- Fixed inverted load math for zero-score R4 teams: previous "pity" only nudged QLab's `loadActionAt` to 10s, which played out 90s of video while the dashboard countdown skipped that team entirely (guarded by `earnedTime > 0`).

## Technical Changes
- `src/server.js` (R4 entry block): mutate `gameState[teamId].earnedTime` and `remainingTime` directly for zero-score teams; logs the gift via `logActivity('system', …)`.
- `sendQLabLoadCue` is now called with the team's true `remainingTime` for both teams, removing the special-case load-position override.
