# v7.3.3 Release Notes — R2 incorrect keeps StreamDeck on current page

**Released: May 2, 2026**

## What's New
- **Round 2 incorrect answer no longer changes the StreamDeck page.** Turn state still advances to the other team (so the round still progresses), but the StreamDeck stays on the current page and the dashboard glow doesn't swap. Operator wanted to keep visual context after a wrong answer instead of being yanked to the opponent's page.

## Technical Changes
- `src/server.js`:
  - New `r2SuppressNextRefreshNav` one-shot flag.
  - `/chart-toppers/incorrect` (R2 branch): adds the answering team to `r2PlayedTeams` (so refresh still advances) and sets the suppress flag.
  - `handleRefreshTracks()`: when the flag is set, runs the turn-advance + R2GO2 retarget but skips `navigateStreamDeck`, `LEADERSD` retarget, and the dashboard glow swap.
  - Flag reset on R2 entry, full round reset, and any R2 correct answer.
