# v7.3.5 Release Notes — Revert v7.3.4 (refreshtracks back to state-driven)

**Released: May 2, 2026**

## What's Reverted
- **v7.3.4 stripped the team-swap and dashboard-glow swap out of `/chart-toppers/refreshtracks` based on a misread instruction.** Restoring the original state-driven `handleRefreshTracks()` behaviour: when one team has played, refresh tracks reloads the same genre AND flips turn state + dashboard glow + StreamDeck to the opponent.

## What's Back
- One team has played → refresh swaps tracks, navigates SD to opponent's page, swaps dashboard glow.
- Both teams have played → refresh retargets R2GO2 → AWO.
- v7.3.3 R2-incorrect "stay on page" suppress flag still in place.
