# v6.0.2 Release Notes — Bundled multi-project Docker updater

## Summary

Adds `scripts/update-dockers.sh` to the repo so any show machine can pull the latest releases of Chart Toppers, Win It In A Minute, and Docforge from GitHub and rebuild their Docker stacks with one command. Named volumes (license, machine ID, persistent settings) are preserved.

## Usage on a new machine

```bash
gh auth login   # one-time, for the private TwistedMelonIO repos

gh api -H "Accept: application/vnd.github.v3.raw" \
  /repos/TwistedMelonIO/chart-toppers/contents/scripts/update-dockers.sh \
  > /tmp/update-dockers.sh
chmod +x /tmp/update-dockers.sh
/tmp/update-dockers.sh           # update all three
/tmp/update-dockers.sh chart     # only chart-toppers
/tmp/update-dockers.sh wim       # only win-it-in-a-minute
/tmp/update-dockers.sh docforge  # only docforge
```

Defaults to cloning into `~/TwistedMelon/<repo>`. Override with `BASE_DIR=...`.
