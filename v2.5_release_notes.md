# Chart Toppers v2.5.0 Release Notes

**Release Date:** 2026-03-02

## 🚀 New Features

### 🗂️ Activity Log Management
- **Reset Activity Logs**: Added secure reset functionality with password protection
- **Confirmation Modal**: Professional modal interface with warning messages
- **Password Verification**: Admin password protection (default: 8888) for sensitive operations
- **Visual Feedback**: Success/error states with button color changes

### 📅 Improved Activity Log Display
- **Today-Only Default**: Activity log now defaults to showing today's entries only
- **Screen Space Optimization**: Reduces clutter by focusing on current day activity
- **Flexible Filtering**: Easy access to 7/30/60 day views when needed
- **Updated Statistics**: Statistics now reflect today's activity by default

### 🔐 Enhanced Security
- **Unified Password System**: All admin functions use consistent password (8888)
- **Configurable Password**: Admin password can be set via ADMIN_PASSWORD environment variable
- **Secure API Endpoints**: Password-protected reset operations with proper validation

## 🛠️ Technical Improvements

### Backend Enhancements
- **New API Endpoint**: `/api/activity/reset` for secure log management
- **Configuration Management**: Centralized admin password in CONFIG object
- **Error Handling**: Comprehensive error responses and user feedback
- **Activity Logging**: Reset actions are logged for audit trail

### Frontend Enhancements
- **Modal System**: Reusable modal components with keyboard support
- **Form Validation**: Real-time password validation with helpful error messages
- **Accessibility**: Escape key support, overlay click to close, proper focus management
- **Responsive Design**: Mobile-optimized modal and button layouts

## 🔧 Bug Fixes & Improvements

### User Experience
- **Consistent Passwords**: Unified 8888 password across all admin functions
- **Better Visual Feedback**: Button states and color changes for user actions
- **Improved Navigation**: Smoother modal animations and transitions
- **Error Messages**: Clear, helpful error messages for failed operations

### Performance
