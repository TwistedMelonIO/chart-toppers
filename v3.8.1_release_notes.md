# v3.8.1 Release Notes — Buzzer Status Indicator

**Released: April 3, 2026**

## What's New

### Buzzer Connection Status in Dashboard
- **Summary**: Dashboard footer now shows whether the QLab Buzzer bridge is connected
- **Details**: The buzzer script sends a heartbeat to the server every 5 seconds. The dashboard displays a green "Connected" or red "Disconnected" indicator in the footer, matching the existing server connection status style.
- **Impact**: Operators can confirm at a glance that buzzers are working before the show starts

## Technical Changes
- Added heartbeat thread to `buzzer/qlab_buzzer.py` (POSTs to `/api/buzzer/heartbeat` every 5s)
- Added `POST /api/buzzer/heartbeat` and `GET /api/buzzer/status` server endpoints
- Added `buzzerStatus` Socket.IO event for real-time connect/disconnect updates
- Added buzzer status indicator in dashboard footer with green/red dot styling
