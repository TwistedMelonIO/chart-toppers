# v7.3.9 Release Notes — R1 iPad is OSC-driven only

**Released: May 2, 2026**

## What's Changed
- **Round 1 iPad no longer auto-advances on correct/incorrect/pass.** The iPad only moves when one of the explicit `/chart-toppers/r1t1` … `/chart-toppers/r1t6` OSCs (added in v7.3.8) arrives. R2/R3/R4 still auto-advance as before.

## Technical Changes
- `src/server.js`:
  - Correct-answer stage host advance: gated to `currentRound !== 1`.
  - Incorrect/pass stage host advance: gated to `currentRound >= 2 && <= 3`.
