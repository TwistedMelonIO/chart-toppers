# v7.3.7 Release Notes — R4 disarms ALL first-team tracks for second team

**Released: May 2, 2026**

## What's Fixed
- **Round 4: when the second team starts, every track the first team played is now disarmed** (not just the most recent). Previously only `R4TG{lastR4TrackPlayed}` was disarmed, so if team A played tracks 5, 9, and 12, tracks 5 and 9 stayed armed and team B could re-play them.

## Technical Changes
- `src/server.js` `/chart-toppers/playing/{team}` OSC handler (R4 second-team branch): replaced single-track disarm with a loop over every entry in `r4TracksPlayed` where the value is `true`, sending `/cue/R4TG{n}/armed 0` for each.
