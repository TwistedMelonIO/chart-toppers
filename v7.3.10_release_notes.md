# v7.3.10 Release Notes — /refreshtracks always navigates SD on team swap

**Released: May 2, 2026**

## What's Fixed
- **R2 `/chart-toppers/refreshtracks` now always switches the StreamDeck to the opponent on a team swap**, even after an R2 incorrect answer. v7.3.3's `r2SuppressNextRefreshNav` flag was suppressing it based on a misread — refreshtracks is always the operator's deliberate "move on" action and should always nav.

## What's Kept
- R2 incorrect still **does not navigate the StreamDeck on its own** (only refreshtracks navigates).
- R2 incorrect still marks the answering team as played in `r2PlayedTeams`, so refreshtracks finds them and runs the team swap.
- v7.3.8 R1 explicit iPad OSC + v7.3.9 R1 auto-advance disable still in place.

## Technical Changes
- `src/server.js`:
  - Removed `r2SuppressNextRefreshNav` flag declaration and all 4 set/reset sites.
  - `handleRefreshTracks()` (single-team-played branch): always navigates, retargets LEADERSD, swaps dashboard glow.
