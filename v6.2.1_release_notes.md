# v6.2.1 Release Notes — Buzzer install adds Homebrew to PATH

## Summary

Fixes `install_service.sh` failing with "gh not found" when run over SSH or other non-interactive shells, even when `gh` is installed. Non-interactive macOS shells use a minimal PATH that excludes `/opt/homebrew/bin`, so `command -v gh` couldn't see it.

## Fix

`install_service.sh` now prepends `/opt/homebrew/bin` (Apple Silicon) and `/usr/local/bin` (Intel) to PATH at the top of the script, so `gh`, `python3`, etc. are findable in all shell contexts.

**Version**: 6.2.1
