# v6.0.0 Release Notes — Chart Toppers 1.0 milestone

## Summary

Chart Toppers reaches its 1.0 milestone. The system is feature-complete for live show use across all four rounds, with QLab 5 + Bitfocus Companion + StreamDeck integration, three full music packs, license-gated activation, persistent settings, and the iPad stage host view.

No further feature work is planned. Future releases (if any) will be patch-level fixes only.

## What's in this release

Carries forward everything shipped through the v5.x line, including:

- **Round flow**: R1 Name It In Five, R2 On Track, R3 Mash Up Mayhem, R4 Final Showdown
- **R4 iPad single-answer mode** (v5.8.0): iPad stays blank on R4 entry and updates per `/chart-toppers/r4/{n}` OSC
- **Music packs**: Pack 1 (UK / USA / Germany), Pack 2 (Europe Med), Pack 3 (Teens) — hot-swappable
- **Buzzer service** (v5.7.x): self-locating, auto-restart, runs as macOS LaunchAgent
- **Genre picker**: split shrink-rate / no-shrink ceiling, locked text colour
- **Resolution picker** in Settings → Advanced
- **License persistence**: named Docker volume, host-side backup of license_key + machine_id
- **OSC bridge**: UDP fire-and-forget + TCP SLIP for QLab bidirectional polling
- **Companion integration**: custom variables for genres, tracks, current team, StreamDeck pages
- **Turn tracking**: coin toss → genre → team A → team B → next genre, R4 loser-first

## Upgrade

```bash
docker compose up -d --build
```

License keys, machine IDs, pack settings, and activity log persist across the rebuild via the `chart-toppers-data` named volume.
