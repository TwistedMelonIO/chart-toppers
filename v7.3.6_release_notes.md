# v7.3.6 Release Notes — R3 always returns StreamDeck to page 12

**Released: May 2, 2026**

## What's New
- **Round 3: any correct or incorrect answer now returns the StreamDeck to page 12.** Previously R3 navigated back to SDE1 (page 1) on each answer, matching R1's flow. Operator wants R3 to always land on the dedicated R3 control page.

## Technical Changes
- `src/server.js`:
  - Correct-answer SD nav (~L913): R1 → page 1 (unchanged), R3 → page 12.
  - Incorrect-answer SD nav (~L3799): R1 → page 1 (unchanged), R3 → page 12.
- All other R3 behaviour (buzzer lockout on first wrong, both-wrong stop + re-arm, track resume, stage host advance) is **unchanged**.
