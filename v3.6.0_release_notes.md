# v3.6.0 Release Notes — Golden Record

**Released: April 3, 2026**

## What's New

### Golden Record Power-Up
- **Summary**: One-time double-points boost per team per game
- **Details**: Each team gets a single Golden Record they can use during Rounds 1-3. When activated via Companion, the next correct answer earns double the seconds for that round. Once used, it's consumed for the rest of the game and resets when the team is reset.
- **Impact**: Adds strategic depth — teams must choose the optimal moment to play their Golden Record for maximum advantage

### How It Works
1. Player verbally requests to use their Golden Record
2. Operator presses the Golden Record button on Companion
3. Dashboard shows pulsing gold indicator (Armed)
4. Next correct answer awards double seconds
5. Golden Record is consumed (greyed out on dashboard)

### Integration Points
- **Companion OSC**: `/chart-toppers/golden-record/anthems` and `/chart-toppers/golden-record/icons`
- **QLab Cues**: `GRA` (Anthems) and `GRI` (Icons) fire on activation
- **REST API**: `POST /api/golden-record/:team`
- **Socket.IO**: `goldenRecord` event

## Technical Changes
- Added `goldenRecordAvailable`, `goldenRecordArmed`, `goldenRecordUsed` to team state model
- Added `activateGoldenRecord()` server function with validation guards
- Added `playGoldenRecordCue()` for QLab integration via OSC bridge
- Added Golden Record indicator UI with armed/used/available states
- Added pulsing gold animation for armed state, greyed-out for used state
- Added `2x` badge on history entries for Golden Record answers
- Fixed stale version number in license gate (was showing v3.4.0)
