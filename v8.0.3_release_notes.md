# Chart Toppers v8.0.3

## Tiebreaker answer on the stage-host iPad

When the R4 tiebreaker fires (DUALGO retargets to TIEBREAK and `tiebreakActive` is set), the stage-host iPad now receives the tiebreaker answer alongside the existing R4 answers.

The tiebreaker track is fixed for this gameshow:

- **Sweet Child O' Mine — Guns N' Roses**

Implementation: in the `/chart-toppers/dualscreen` handler, when the tie is detected, the answer is appended to `stageHostAnswers` and the iPad's `currentTrack` index is advanced to it. The round label stays as `Final Round` (no separate "Tie Breaker" label) so it reads continuously with R4.

The answer text is hardcoded in `src/server.js` because the tiebreaker track is the same every show.

## Files changed
- `src/server.js` — added the tiebreaker answer push in `/dualscreen` (~6 lines).
