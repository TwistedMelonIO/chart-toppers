# v6.2.2 Release Notes — Bulk buzzer deploy script

## Summary

Adds `scripts/deploy-buzzer.sh` — push the prebuilt buzzer .app to one or many show machines from the dev machine over SSH. No GitHub auth needed on the targets, no Homebrew/PyInstaller, no per-machine builds.

## Usage

From the dev machine (the one that built the .app):

```bash
./scripts/deploy-buzzer.sh user@host [user@host ...]
```

Per target it:
1. Verifies SSH connectivity.
2. SCPs `QLab Buzzer.app.zip` to `/tmp/`.
3. Unzips to `~/Applications/QLab Buzzer.app`.
4. Writes the LaunchAgent (using `open -W -a` so Accessibility persists).
5. Loads the LaunchAgent.
6. Prints the one-time Accessibility step for that Mac.

After deploy, on each target Mac add `~/Applications/QLab Buzzer.app` to **Privacy & Security → Accessibility** once. Same binary on every Mac means the grant survives every future routine update.

## Prerequisites on each target

- Remote Login enabled (System Settings → General → Sharing).
- Your SSH key authorised: `ssh-copy-id user@host`.
