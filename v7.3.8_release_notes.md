# v7.3.8 Release Notes — R1 explicit iPad track OSC

**Released: May 2, 2026**

## What's New
- **6 new OSC addresses for Round 1: `/chart-toppers/r1t1` … `/chart-toppers/r1t6`.** Each one explicitly sets the stage-host iPad to that track number. Designed to be fired from inside each track's QLab group cue so the iPad always reflects the actual track playing — no more guessing or drift from auto-advance.
- Auto-advance on correct/incorrect/pass is **unchanged** for now (operator will tackle disabling it later if needed).

## Technical Changes
- `src/server.js` OSC handler block: new regex `/^\/chart-toppers\/r1t([1-6])$/`. When matched in R1 with answers loaded, sets `stageHostAnswerIndex = trackNum - 1` and emits `currentTrack { trackNumber, total }`. Ignores out-of-range / wrong-round / no-answers-loaded.
