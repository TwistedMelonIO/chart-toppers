# v7.1.0 Release Notes — Golden Record Sting (GBUZZER)

**Released: May 2, 2026**

## What's New

### Golden Record Sting Cue
- **Summary**: Activating Golden Record via OSC now also fires `/cue/GBUZZER/start` to QLab.
- **Details**: When Companion sends `/chart-toppers/golden-record/anthems` or `/chart-toppers/golden-record/icons`, the server arms the team's Golden Record (existing behaviour) and additionally triggers the `GBUZZER` cue in QLab for an audible sting.
- **Impact**: Operators get a sound cue the moment Golden Record is armed from the StreamDeck — no extra Companion button needed.

## Technical Changes
- `src/server.js`: shared OSC handler for `/chart-toppers/golden-record/(anthems|icons)` now calls `sendBridgeOsc('/cue/GBUZZER/start', null, …)` after a successful activation.
